#  E-Commerce Customer Behavior Dashboard

Interactive Power BI dashboard analyzing customer behavior 
of an e-commerce platform (350 customers).

## Questions Explored
- Which city generates the most revenue?
- What is the gender distribution of customers?
- Which age group purchases the most items?
- Do Gold/Silver/Bronze members spend differently?

## Tools
Power BI Desktop

## Dataset
E-Commerce Customer Behavior dataset (350 customers, 11 columns).
Source: [Kaggle](https://www.kaggle.com/datasets/uom190346a/e-commerce-customer-behavior-dataset)

## Key Insights
| KPI | Value |
|-----|-------|
| Total Customers | 350 |
| Avg. Total Spend | $16,540 |
| Avg. Items Purchased | 12.60 |
| Top City | Chicago |
| Gender Split | 50% Female / 50% Male |
| Top Membership | Gold |

## Dashboard Visuals
- KPI Cards — Total customers, avg. spend, avg. items purchased
- Bar Chart — Revenue by city
- Donut Chart — Customer distribution by gender
- Line Chart — Avg. items purchased by age
- Matrix — Avg. spend by gender and membership type

## How to Open
1. Download Power BI Desktop (free) from powerbi.microsoft.com
2. Download the dataset from Kaggle (link above)
3. Open `ecommerce-dashboard.pbix`
